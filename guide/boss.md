# Boss战斗机制

修仙服务器对Boss进行了全面强化——动态缩放、专属弹幕、魔改技能，让每场Boss战都充满挑战。

## Boss动态缩放

所有Boss根据挑战者的修仙等级动态调整强度：

- **启用**：<i class="fa-solid fa-check"></i>
- **每级HP加成**：+5%
- **每级伤害加成**：+5%
- **Boss喊话间隔**：10秒
- 修仙等级越高，Boss越强！

### 缩放公式

```
实际HP  = Boss基础HP  × (1 + 修士修仙等级 × 0.05)
实际伤害 = Boss基础伤害 × (1 + 修士修仙等级 × 0.05)
```

## 怪物吸血机制

- **普通怪物吸血**：伤害的 **20%** 转化为生命
- **Boss吸血**：伤害的 **35%** 转化为生命
- **吸血上限**：基础生命的 **×1.5**

> <i class="fa-solid fa-lightbulb"></i> 输出越高，怪物回血越快！建议优先击杀小怪。

## 修士减伤

启用修士减伤机制，减少来自怪物的伤害，让修仙者在面对高强度Boss时更具生存力。

## Boss战策略

### 通用建议
1. **准备充足药水** — Boss血量动态缩放，战斗时间可能很长
2. **带好治疗丹/恢复丹** — 关键时刻保命
3. **注意弹幕躲避** — Boss弹幕速度最高12.0（月总加强），走位很关键
4. **优先击杀小怪** — 怪物有吸血，小怪不清理Boss会持续回复
5. **组队作战** — 带新人还能赚功德

### 各Boss注意点
- **克苏鲁之眼**：55%血以下弹幕数量翻倍，需提高警惕
- **史莱姆王**：45%血召唤小怪，需快速清场
- **骷髅王**：50%血触发雷暴，注意躲避
- **双子魔眼**：二阶段12弹幕的流星坠落极难躲避
- **月总加强**：6秒间隔的全图案弹幕+雷暴，全服最强Boss
