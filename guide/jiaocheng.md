# 指令教程

从零开始，手把手教你使用修仙服务器的全部指令。

---

## 指令速查表

### 所有玩家指令

| 主命令 | 缩写/别名 | 说明 |
|--------|----------|------|

| `/功法 释放 <ID>` | — | 释放指定功法技能 |

## 第一章：初入修仙（第一天）

### 1.1 觉醒灵根

进入服务器后，系统会自动检测你的灵根资质。输入以下命令查看：

```
/xx
```

你会看到类似这样的面板：

```
══════════ 修仙面板 ══════════
境界：凡人
灵根：火灵根
灵气：100 / 100
修为：0 / 150
功法：无
══════════════════════════════
```

### 1.2 获取灵气

击杀怪物是获取灵气的主要方式。每造成 100 伤害可获得约 5 灵气。去野外打几个史莱姆或僵尸攒灵气。

### 1.3 首次突破

灵气攒到一定程度后，尝试突破到练气期：

```
/tp
```

> 💡 突破消耗当前灵气的 70%，成功率 37.5%（练气1阶基础值）。失败了别灰心，继续打怪攒灵气重试！

```
# 如果用 /xx 查看显示灵气够了但一直失败，
# 连续失败3次后会触发保护，提高成功率。
```

## 第二章：修炼进阶（第二天）

### 2.1 开始修炼

有了练气期基础后，可以开始挂机修炼：

```
/xl
```

系统会提示「开始修炼」，你的角色会进入定身状态。每 10 秒聊天栏会显示进度。

修炼期间每秒消耗 5 灵气，12% 转化为修为。想要停止修炼再次输入 `/xl`。

### 2.2 查看功法

输入以下命令查看已解锁的功法：

```
/gf
```

### 2.3 学习功法

达到对应境界+小阶后，可以学习功法：

```
/gf 可学         # 查看可学功法列表
/gf 学习 1       # 学习ID为1的功法（灵气弹）
```

### 2.4 释放功法

学习后使用功法技能：

```
/gf 释放 1       # 释放灵气弹
```

> ⚠️ 功法有全局冷却5秒！释放后需要等5秒才能用下一个。

## 第三章：炼丹制药

### 3.1 查看丹药库存

```
/ld
```

### 3.2 炼制突破丹

突破丹能大幅提高突破成功率：

```
/ld 突破        # 自动选择能炼制的最高品质
```

炼制需要消耗灵气和修为（中等品质以上），且高级/极品有失败受伤风险。建议在修仙等级较高时挑战极品丹药。

### 3.3 炼制修炼丹

修炼丹可以加速修炼效率（持续90秒）：

```
/ld 修炼
```

> 🌊 修炼丹 + 灵气潮汐事件 = 修炼效率 ×2.25！

### 3.4 疗伤

战斗中受伤了？用疗伤丹回血：

```
/ld 疗伤        # 炼制疗伤丹
/ld 使用 疗伤   # 使用疗伤丹
```

## 第四章：突破攻略

### 4.1 带丹突破

突破时可以自动使用突破丹（前提是你有库存）：

```
/tp 是          # 使用突破丹辅助突破
```

### 4.2 大境界突破流程

```
1. /xx          # 确认当前是第9小阶
2. /gf 释放 X   # 击杀所需Boss（详见境界页面）
3. /ld 突破     # 炼制突破丹
4. /tp 是       # 带丹突破
5. 金丹及以上需渡劫——扛住天雷！
```

### 4.3 渡劫要点

- 金丹起每次大境界突破都要渡劫
- 天雷数量和伤害随境界递增（飞升18道每道400伤）
- 天道境界越高减免越多（最高-40%）
- 务必提前备好渡劫丹和回复药水

## 常见问题

### Q: `/tp` 和 Terraria 的传送指令冲突吗？
不冲突。修仙插件的 `/tp` 是 `/突破` 的缩写，和原版 `/tp` (teleport) 是不同的命令前缀。

### Q: 修炼时能移动吗？
不能。修炼期间会定身，再次输入 `/xl` 停止修炼即可恢复移动。

### Q: 突破一直失败怎么办？
连续失败3次会触发保护机制，成功率会提高。同时建议：
1. 使用突破丹 `/tp 是`
2. 等待天道试炼事件（+15%成功率）
3. 提升天道境界获得额外加成

### Q: 怎么看自己的灵根？
输入 `/xx` 打开修仙面板，会显示灵根类型。

### Q: 功法ID怎么看？
输入 `/gf 可学` 查看可学功法列表，每门功法都有对应ID。
