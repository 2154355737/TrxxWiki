# 算法公开

本文档公开服务器所有核心算法的计算公式，帮助玩家精确规划修仙路线。

---

## 一、突破成功率

### 小阶突破

```
基础成功率 = 0.5（全局基础） × 境界小阶突破成功率

最终成功率 = 基础成功率
            + 灵根突破加成
            + 突破丹效果值
            + 天道境界突破加成
            + 天道试炼事件加成（若激活）
            + 连续失败保护（若触发）
```

各境界小阶成功率（来自 `修仙境界.json`）：

| 境界 | 小阶成功率 | 叠加基础(0.5)后 |
|------|-----------|-----------------|
| 练气 | 75% | 37.5% |
| 筑基 | 65% | 32.5% |
| 金丹 | 55% | 27.5% |
| 元婴 | 45% | 22.5% |
| 化神 | 38% | 19.0% |
| 渡劫 | 30% | 15.0% |
| 大乘 | 25% | 12.5% |
| 飞升 | 18% | 9.0% |

### 大境界突破

```
基础成功率 = 0.25（全局基础） × 境界大境界突破成功率

最终成功率 = 基础成功率 + 同上各项加成
```

各境界大境界成功率：

| 境界 | 大境界成功率 | 叠加基础(0.25)后 |
|------|-------------|------------------|
| 练气→筑基 | 45% | 11.25% |
| 筑基→金丹 | 40% | 10.0% |
| 金丹→元婴 | 30% | 7.5% |
| 元婴→化神 | 25% | 6.25% |
| 化神→渡劫 | 20% | 5.0% |
| 渡劫→大乘 | 15% | 3.75% |
| 大乘→飞升 | 12% | 3.0% |

### 突破成功率计算示例

**场景**：天灵根（+20%）修士在化神9阶，使用极品突破丹（+65%），天道境界为「代天行道」（+8%），突破到大乘期。

```
基础成功率 = 0.25 × 0.12 = 3.0%
灵根加成  = +20.0%
突破丹    = +65.0%
天道加成  = +8.0%
─────────────────────
最终成功率 = 3.0 + 20 + 65 + 8 = 96.0%
```

### 突破成本

```
突破灵气消耗 = 当前灵气 × 境界突破消耗比例
```

| 境界 | 消耗比例 |
|------|---------|
| 练气 | 70% |
| 筑基 | 75% |
| 金丹 | 80% |
| 元婴 | 82% |
| 化神 | 85% |
| 渡劫 | 88% |
| 大乘 | 90% |
| 飞升 | 95% |

## 二、渡劫计算

### 天雷伤害

渡劫时每道天雷造成的伤害由境界配置决定：

| 境界 | 天雷道数 | 每道伤害 | 间隔 |
|------|---------|---------|------|
| 金丹 | 3道 | 30 | 2.5秒 |
| 元婴 | 6道 | 60 | 2.5秒 |
| 化神 | 9道 | 100 | 2.0秒 |
| 渡劫 | 12道 | 160 | 2.0秒 |
| 大乘 | 15道 | 250 | 1.5秒 |
| 飞升 | 18道 | 400 | 1.5秒 |

### 渡劫减伤

```
实际天雷伤害 = 基础天雷伤害 × (1 - 天道渡劫减伤)
```

| 天道境界 | 减伤 |
|----------|------|
| 普通人 | 0% |
| 有德 | 0% |
| 积善 | 10% |
| 天道眷顾 | 20% |
| 代天行道 | 30% |
| 天人合一 | 40% |

**示例**：天人合一修士渡飞升劫 → 400 × (1 - 0.4) = **240伤害/道**（18道共 4,320 总伤害）

### 渡劫失败

```
渡劫失败扣血 = 150（固定）
渡劫掉落物品 = true（可能损失道具）
```

## 三、炼丹成功率

```
丹药炼制成功率 = 丹药品质基础成功率 + 修仙等级 × 1%
```

| 修仙等级 | 下阶突破丹(70%) | 极品渡劫丹(20%) |
|----------|----------------|-----------------|
| 等级10 | 80% | 30% |
| 等级30 | 100% | 50% |
| 等级50 | 100%（封顶） | 70% |
| 等级80 | 100% | 100%（封顶） |

### 炼丹失败伤害

| 品质 | 突破丹 | 渡劫丹 | 治疗丹 | 恢复丹 |
|------|--------|--------|--------|--------|
| 下阶 | 0 | 0 | 0 | 0 |
| 中等 | 0 | 0 | 0 | 0 |
| 高级 | 30 | 35 | 25 | 28 |
| 极品 | 80 | 100 | 60 | 70 |

> ⚠️ 极品渡劫丹失败直接扣100血！务必在满血状态下炼制。

## 四、功法威力计算

```
实际伤害 = 功法基础伤害 × 威力倍率 × 灵根契合倍率 × 天灵根全功法倍率
```

### 灵根契合倍率

| 条件 | 倍率 |
|------|------|
| 灵根 ≠ 功法契合灵根 | ×1.0 |
| 灵根 = 功法契合灵根 | ×1.5 |
| 天灵根所有功法 | ×1.2 |
| 天灵根 + 同灵根功法 | ×1.5 × 1.2 = ×1.8 |

**示例**：天灵根修士释放「飞升羽化」（天灵根契合）：
- 同灵根 ×1.5 + 所有功法 ×1.2 = **×1.8 倍率**

---

## 五、防御减伤

```
最终防御 = 基础防御 × 灵根防御倍率 + 功法/Buff额外防御 + 每小阶防御加成
```

- 土灵根防御 ×1.15
- 天灵根防御 ×1.1
- 每小阶 +1 防御

---

## 六、速度计算

```
实际速度 = 基础速度 × Π(每大境界速度倍率) × 灵根无额外倍率
```

每突破1个大境界，速度 ×1.02：
- 练气 → 飞升（8次大突破）：速度 × 1.02⁸ = **×1.172**

---

## 七、吸血公式

```
怪物吸血量 = 造成伤害 × 吸血比率
最终吸血量 = min(吸血量, 怪物基础HP × 1.5)
```

| 类型 | 吸血比率 | 上限倍率 |
|------|---------|---------|
| 普通怪物 | 20% | ×1.5 |
| Boss | 35% | ×1.5 |

---

## 八、Boss动态缩放

```
实际HP  = Boss基础HP  × (1 + 修士修仙等级 × 0.05)
实际伤害 = Boss基础伤害 × (1 + 修士修仙等级 × 0.05)
```

**示例**：修仙等级 50 的修士挑战克苏鲁之眼（基础HP 2800）：
- 实际HP = 2800 × (1 + 50 × 0.05) = 2800 × 3.5 = **9,800 HP**

受缩放的Boss完整列表见 [Boss战斗](/guide/boss) 页面。

---

## 九、修炼效率公式

```
每秒修为 = 修炼消耗灵气(5.0) × 灵气转修为比例(0.12) = 0.6

每秒灵气净变化 = 灵气恢复 - 修炼消耗(5.0)
```

灵气恢复不足时无法持续修炼，需要等待自然恢复或使用恢复丹。

### 灵气潮汐期间

```
修炼每秒修为 = 0.6 × 1.5 = 0.9
灵力恢复 × 2.0
```

## 十、功德衰减

```
每日衰减 = 当前功德 × 5%（不活跃7天后触发）
```

例如：1000功德，不活跃7天后：
- 第8天：1000 - 50 = 950
- 第9天：950 - 47.5 = 902.5
- 以此类推...

---

## 十一、决斗死亡惩罚

```
决斗失败扣灵气 = 当前灵气 × 20%
```

比普通死亡的25%略低，但仍有不小代价。
