## 突破机制详解

### 成功率计算公式

突破成功率由**全局基础概率 × 境界配置概率**叠加各项加成：

```
小阶突破:
基础成功率 = 0.5（全局基础） × 境界小阶突破成功率

大境界突破:
基础成功率 = 0.25（全局基础） × 境界大境界突破成功率

最终成功率 = 基础成功率 + 灵根加成 + 突破丹加成 + 天道境界加成 + 事件加成
```

> <i class="fa-solid fa-ruler-triangle"></i> 完整计算公式见 [算法公开](/guide/suanfa)。

| 筑基 | 0.5 × 0.65 = **32.5%** | 0.25 × 0.40 = **10.0%** |
| 金丹 | 0.5 × 0.55 = **27.5%** | 0.25 × 0.30 = **7.5%** |
| 元婴 | 0.5 × 0.45 = **22.5%** | 0.25 × 0.25 = **6.25%** |
| 化神 | 0.5 × 0.38 = **19.0%** | 0.25 × 0.20 = **5.0%** |
| 渡劫 | 0.5 × 0.30 = **15.0%** | 0.25 × 0.15 = **3.75%** |
| 大乘 | 0.5 × 0.25 = **12.5%** | 0.25 × 0.12 = **3.0%** |
| 飞升 | 0.5 × 0.18 = **9.0%** | 0.25 × 0.08 = **2.0%** |

### 突破丹叠加

突破丹的基础加成为 **+25%**（全局配置 `突破丹加成: 0.25`），此为基础保底加成。

使用丹药可在此之上额外叠加：

| 突破丹品质 | 额外加成 | 合计加成 |
|-----------|---------|---------|
| 不使用 | +0% | +25% |
| 下阶 | +20% | +45% |
| 中等 | +30% | +55% |
| 高级 | +45% | +70% |
| 极品 | +65% | +90% |

### 小阶突破
- 每个大境界包含 **9个小阶**（练气1阶 → 练气9阶）
- 消耗当前灵气的一定比例即可尝试突破
- 基础成功率随境界提升而递减
- 全局突破冷却：**5秒**（每次突破后等待）

### 大境界突破
- 从练气9阶突破至筑基1阶，以此类推
- **必须先击杀对应Boss** 才能解锁大境界突破
- 金丹及以上境界需要 **渡劫**，承受天雷轰击
- 突破期间会被 **定身**（3秒），无法移动

### 连续失败保护
- 连续突破失败 **3次** 后自动触发保护机制
- 触发后获得 **10功德** 作为补偿
- 保护机制提高后续突破成功率

### 渡劫系统
- 金丹起每道天雷间隔 2.5 秒，大乘后缩短至 1.5 秒
- 渡劫失败扣血 **150**
- 渡劫时会掉落物品
- **天道境界**越高，渡劫减伤越多（最高 **40%**）
- 渡劫丹可额外提升渡劫成功率

### 属性成长（全局配置）

每次小阶突破获得的属性加成（累加）：

| 属性 | 每小阶加成 |
|------|-----------|
| HP上限 | +10 |
| 防御 | +1 |
| 魔力上限 | +1 |

每次大境界突破速度倍率 **×1.02**（累乘）：
- 练气→飞升（8次大突破）：速度累计 ×1.02⁸ ≈ **×1.172**
