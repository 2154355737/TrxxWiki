# 天道系统

天道系统是修仙服务器的**善恶/功德**机制。积德行善可获得功德值，提升天道境界，享受各种增益。反之，恶意行为会受到天道惩罚。

---

| 境界 | 功德阈值 | 突破加成 | 渡劫减伤 | 灵气倍率 | 功法倍率 | 商店折扣 |

> 🌟 天道境界越高，突破越容易、渡劫越安全、灵气恢复越快、天道商店越便宜！

| 🏆 Boss服务器首杀 | +100 | 全服第一个击杀该Boss |
| 🎯 Boss个人首杀 | +50 | 个人首次击杀该Boss |
| ⬆️ 大境界突破 | +30 | 每次成功突破大境界 |

## 天道事件

天道会随机触发全服事件，影响所有在线玩家：

### 灵气潮汐 🌊

| 属性 | 值 |
|------|-----|
| 触发权重 | 30（相对权重，较常见） |
| 持续时间 | 30分钟 |
| 灵气恢复倍率 | **×2.0** |
| 修炼灵气倍率 | **×1.5** |
| 突破成功率加成 | +0% |
| 突破冷却减半 | ❌ |
| 渡劫伤害倍率 | ×1.0 |
| 击杀功德倍率 | ×1.0 |

**开场广播**：【天道】灵气潮汐涌动，天地灵机大盛！

**结束广播**：【天道】灵气潮汐渐息。

> 🌊 灵气潮汐是修炼的最佳时机！灵气恢复翻倍+修炼效率×1.5，配合修炼丹效果更佳。

### 天道试炼 ⚡

| 属性 | 值 |
|------|-----|
| 触发权重 | 20（相对权重） |
| 持续时间 | 30分钟 |
| 灵气恢复倍率 | ×1.0 |
| 修炼灵气倍率 | ×1.0 |
| 突破成功率加成 | **+15%** |
| 突破冷却减半 | ✅ **是** |
| 渡劫伤害倍率 | **×0.8**（减免20%） |
| 击杀功德倍率 | ×1.0 |

**开场广播**：【天道】试炼开启，突破屏障暂时松动。

**结束广播**：【天道】试炼结束，天机复归。

> 🎯 天道试炼是突破的最佳时机！+15%成功率、冷却减半、渡劫减伤20%——三重利好叠加！

### 事件触发机制
- **随机间隔**：120分钟～360分钟（2~6小时）
- **全服广播**：所有玩家可见
- **事件随机**：灵气潮汐（权重30）vs 天道试炼（权重20）

## 功德衰减

如果玩家长时间不活跃，功德值会逐渐衰减：

- **衰减启用**：✅ 是
- **不活跃天数**：7天
- **衰减比例**：每日衰减当前功德的 5%

> ⚠️ 超过7天不登录，功德值开始每日递减。保持活跃才能维持天道境界！

## 天道策略建议

### 快速提升天道境界
1. **优先击杀Boss** — 首杀奖励最多（个人+50，服务器首杀+100）
2. **组队带新人** — 带低境界玩家打Boss，每人 +20
3. **积极突破** — 每次大境界突破 +30
4. **参与决斗** — 胜利 +5，积少成多
5. **炼制极品丹药** — 每次成功 +15

### 避免功德惩罚
- ❌ **不要击杀低境界玩家** — 境界差 ≥ 2 即触发 -200 惩罚
- ✅ 同境界或更高境界的PVP不受惩罚
