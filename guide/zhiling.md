# 指令大全

修仙服务器全部指令速查，含别名（缩写）。

## 玩家指令（含缩写）

| 主命令 | 缩写 | 说明 |
|--------|------|------|
| `/修仙` | `/xx` | 打开修仙主面板：境界、灵气、灵根、丹药、功法、属性 |
| `/突破 [是/否]` | `/tp` | 尝试突破境界；加 `是` 自动使用突破丹 |
| `/修炼` | `/xl` | 开始/停止修炼（定身挂机，每秒耗5灵气） |
| `/功法` | `/gf` | 查看已学习功法列表 |
| `/功法 可学` | — | 查看当前可学功法 |
| `/功法 学习 <ID>` | — | 学习指定ID的功法 |
| `/功法 释放 <ID>` | — | 释放指定ID的功法技能 |
| `/炼丹` | `/ld` | 查看丹药库存 |
| `/炼丹 突破` | — | 炼制突破丹 |
| `/炼丹 修炼` | — | 炼制修炼丹 |
| `/炼丹 疗伤` | — | 炼制疗伤丹 |
| `/炼丹 使用 疗伤` | — | 使用疗伤丹回血 |
| `/修仙排行` | `/xxph` | 在线修仙者排行榜 Top10 |

## 缩写对照

老玩家用缩写速度翻倍：

| 完整 | 缩写 | 记忆方法 |
|------|------|----------|
| `/修仙` | `/xx` | 修→X, 仙→X |
| `/突破` | `/tp` | 突→T, 破→P |
| `/修炼` | `/xl` | 修→X, 炼→L |
| `/功法` | `/gf` | 功→G, 法→F |
| `/炼丹` | `/ld` | 炼→L, 丹→D |
| `/修仙排行` | `/xxph` | 修仙→XX, 排行→PH |

## 管理指令（服主）

| 命令 | 说明 |
|------|------|
| `/修仙管理 帮助` | 查看管理命令帮助 |
| `/修仙管理 设置境界 <玩家> <境界> [小阶]` | 设置玩家境界 |
| `/修仙管理 设置灵气 <玩家> <数值>` | 设置玩家灵气 |
| `/修仙管理 设置灵根 <玩家> <灵根>` | 设置玩家灵根 |
| `/修仙管理 查看 <玩家>` | 查看玩家修仙数据 |
| `/修仙管理 重置 <玩家>` | 重置玩家修仙数据 |
| `/修仙管理 保存` | 立即保存所有数据 |
| `/reload` | 热加载配置文件 |

灵根可选：`金` / `木` / `水` / `火` / `土` / `天`

境界可选：`凡人` `练气` `筑基` `金丹` `元婴` `化神` `渡劫` `大乘` `飞升`

## 聊天显示

玩家聊天自动带境界前缀，无需额外指令：

```
[练气3阶]玩家名: 消息内容
[金丹7阶]玩家名: 消息内容
```

## 其他机制

| 操作 | 说明 |
|------|------|
| 击杀怪物 | 伤害的5%转化为灵气 |
| 击杀Boss | 获得功德（个人首杀+50，服务器首杀+100） |
| 普通死亡 | 扣除25%当前灵气 |
| 决斗死亡 | 扣除20%当前灵气 |
| 突破成功 | 自动获得装备和Buff奖励 |
| 连续突破失败3次 | 触发保护 + 获得10功德 |
| 功法全局冷却 | 5秒 |
| 炼丹冷却 | 60秒 |
