# 决斗系统

修仙者之间的切磋较量，胜利可获得功德和荣誉，失败则要承受一定代价。

---

## 决斗参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 决斗启用 | ✅ | 系统已启用 |
| 决斗冷却 | 120秒 | 发起决斗后冷却2分钟 |
| 最低境界 | 练气 | 凡人无法参与决斗 |
| 决斗超时 | 300秒（5分钟） | 超时未分胜负自动结束 |
| 逃跑距离 | 120格 | 超出此距离视为逃跑 |
| 挑战响应超时 | 30秒 | 被挑战者需在30秒内回应 |
| 允许赌注 | ✅ | 可以押注物品 |
| 赌注上限 | 10个 | 最多押注10个物品 |
| 死亡扣灵气 | 20% | 决斗失败扣除当前灵气的20% |

---

## 决斗流程

1. **发起挑战** — 使用 `/juedou <玩家名>` 向其他修士发起决斗
2. **等待响应** — 被挑战者有 **30秒** 时间接受或拒绝
3. **进入战斗** — 双方开始切磋，限时 **5分钟**
4. **分出胜负** — 一方死亡或逃跑则结束
5. **结算奖励** — 胜者获得功德 +5，败者扣除20%灵气

---

## 决斗规则

### 胜利条件
- 击杀对方
- 对方逃跑（超出120格范围）
- 对方超时未响应（自动判胜）

### 失败代价
- 扣除当前灵气的 **20%**（比普通死亡25%略低）
- 如果设了赌注，赌注归胜者所有

### 逃跑判定
- 决斗中任一方离开超过 **120格** 即判定逃跑
- 逃跑方视为失败，承受失败惩罚

---

## 功德获取

| 行为 | 功德 |
|------|------|
| 决斗胜利 | +5 |
| 恶意击杀低境界玩家 | **-200**（境界差≥2） |

> ⚠️ 公平决斗可以获得功德。但如果恶意击杀低于自己2个大境界以上的玩家，会受到天道严惩！

---

## 决斗策略

### 战前准备
1. 确保灵气充足（功法需要灵气）
2. 检查功法冷却状态
3. 带好治疗丹/恢复丹

### 功法选择
- 防御型功法开局（护体罡气/金钟罩等）先保证生存
- 利用身法（缩地成寸）灵活走位
- 高威力攻击功法找时机爆发

### 灵根克制
- 土灵根（高防御）克制火灵根（纯输出）
- 木灵根（高恢复）适合持久战
- 天灵根全属性优势，但觉醒概率仅5%

## 赌注系统

决斗允许押注物品作为彩头：

```
/juedou <玩家名> <赌注物品> <数量>
```

- 最多押注 **10个** 物品
- 胜者获得全部赌注
- 败者失去赌注 + 20%灵气

>
